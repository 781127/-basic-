# -basic-
管理者の登録とログイン、および学生データに対するバックエンドでの追加・削除・更新・照会機能のみを含む。

It only includes administrator registration and login, as well as backend functions to add, delete, modify, and query student data.

仅包含管理员注册与登录，后台对学生数据进行增删改查功能
